# YesYesGram
Nonogram solver with Selenium (backtracking algorithm). Still somewhat slow (~1min for 20x20 nonogram) but hey,
it's twenty times faster than me.

Solves nonograms on [nonograms.org](http://nonograms.org/)

## Dependencies
- `selenium`
- `webdriver-manager`
- `rich`
- `tqdm`

## Usage
- `python3 main.py`
